HTML Tags

  1. HTML
    • The html tag is the root element of an HTML document.
    • It contains all other HTML elements on the page.

  2. head
    • The head tag is used to define the head section of an HTML document.
    • It contains meta-information about the document, such as the title, charset, and linked stylesheets.

  3. title
    • The title tag is used to specify the title of the HTML document.
    • The text inside this tag is displayed in the browser's title bar or tab.

  4. meta
    • The meta tag is used to provide metadata about the HTML document.
    • It includes attributes like charset, viewport, and description.

  5. link
    • The link tag is used to link external resources to the HTML document.
    • It's commonly used for linking stylesheets (rel="stylesheet") and icons (rel="icon").

  6. script
    • The script ag is used to embed or reference JavaScript code in the HTML document.
    • It can be placed in the head or at the end of the body

  7. script
    • The script tag is used to embed or reference JavaScript code in the HTML document.
    • It can be placed in the head or at the end of the body

  8. style
    • The style tag is used to define CSS styles directly within the HTML document.
    • It can be placed in the head section.

  9. body
    • The body tag contains the visible content of the HTML document.
    • It includes text, images, links, and other elements displayed in the browser.

  10. <h1> <h2> <h3> <h4> <h5> <h6>
    • These are heading tags used to define the hierarchy of headings on a page.
    • h1 represents the highest level, while h6 represents the lowest..

  11. <p>
    • The p tag is used to define paragraphs of text.
    • It creates a block-level element with space before and after the content.

  12. <a>
    • The atag is used to create hyperlinks to other web pages or resources.
    • It includes an href attribute specifying the URL..

  13. <img>
    • The imgtag is used to embed images in an HTML document.
    • It requires the src attribute to specify the image source.

  14. <div>
    • The div tag is a block-level container used for grouping and styling elements.
    • It has no inherent semantic meaning.

  15. <form>
    • The form tag defines various types of input fields within a form.
    • Common types include text, password, checkbox, radio, and more.
