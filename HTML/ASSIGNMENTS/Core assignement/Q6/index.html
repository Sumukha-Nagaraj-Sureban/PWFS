1. HTML (HyperText Markup Language):

    HTML is the foundation of web development, used for structuring the content of web pages.

    Tech Stack Name Primary use Cases Key Features/Benifits
    HTML Defining the structure of web pages, improving the SEO creating content hierarchy, and marking up elements for browsers
    to render.
    • Easy to learn and use for basic web content creation.
    • Supports multimedia elements like images and videos.
    • Semantic elements enhance SEO and accessibility.
    • Compatible with all major browsers. |

  2. CSS (Cascading Style Sheets):

    CSS is used for styling and layout of web pages, making them visually appealing.

    Tech Stack Name Primary use Cases Key Features/Benifits
    CSS Defining the presentation, layout, and design of web pages, ensuring consistency in appearance across different
    devices.
    • Separates content from presentation, improving maintainability.
    • Provides flexibility in designing responsive layouts.
    • Supports animations and transitions.
    • Extensible through CSS frameworks like Bootstrap.

  3. JavaScript

    JavaScript is a dynamic scripting language used for adding interactivity and behavior to web pages..

    Tech Stack Name Primary use Cases Key Features/Benifits
    JavaScript: Enhancing user experience by enabling the dnamic content, from validation, real-timeupdates,
    and more on web pages
    • Supports both front-end (browser) and back-end (Node.js) development.
    • Extensive library ecosystem (e.g., React, Angular, Vue.js).
    • Asynchronous capabilities (Promises, Async/Await) for smooth user interactions.
    • Cross-browser compatibility.
